# Hot Springs AC & Heating

A standalone static HVAC website ready for GitHub and Cloudflare Pages.

## Preview locally

From this folder:

```bash
python3 -m http.server 8081
```

Open `http://localhost:8081`.

From the parent workspace server, preview at:

```text
http://127.0.0.1:8080/outputs/hvac-site/
```

## Before publishing

1. Replace the placeholder phone number, reviews, rating and service areas.
2. Confirm business address, license, insurance, warranties, financing and emergency-service claims.
3. Connect the request form to a CRM or form service.
4. Update the domain in canonical, Open Graph, schema, sitemap and robots if needed.

## Cloudflare Pages

Use no framework preset, no build command and `/` as the output directory when this folder is the repository root.
